# vue-crud-example

Exemplo de CRUD frontend em Vue.js consumindo API REST.

## O que demonstra
- Componentização e reuso em Vue.
- Listagem, criação, edição e exclusão consumindo API REST.
- Roteamento com Vue Router e gerência de estado básica.

## Como rodar
```bash
npm install
npm run serve
```
